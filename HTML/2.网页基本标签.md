# 网页基本标签

## 一，标题标签

```html
<h1>一级标签</h1>
<h2>二级标签</h2>
<h3>三级标签</h3>
```

## 二，段落标签

```html
<p>分段标签</p>
```

## 三，换行标签

```html
换行标签<br/><!--无论换多少行都属于一个段落-->
```

## 四，水平线标签

```html
<hr/ 属性="属性值">
```

## 五，字体样式标签

```html
<strong>粗体</strong> <b>加粗</b>
<em>斜体</em>         <i>斜体</i>
<del>删除线</del>     <s>删除线</s>
<ins>下划线</ins>     <u>下划线</u>
```

## 六，注释和特殊符号

```html
空格：&nbsp;<!--&后有很多特殊符号-->
```

## 七，文本控制标记

```html
align="center"(默认)(left)(right)<!--设置对齐方式-->
size="2px"(默认)<!--设置粗细-->
color="#RGB"(rgb(r,g,b))(颜色英文名称)<!--设置颜色-->
width="100%"(默认)(npx)<!--设置宽度-->
```
