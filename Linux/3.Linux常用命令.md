## 打包压缩命令

### tar

<img src="img/3.Linux常用命令/image-20230531112434916.png" alt="image-20230531112434916" style="zoom:80%;" />

> 打包：
>
> ```
> tar -cvf xxx.tar xxx
> ```
>
> 打包并压缩：
>
> ```
> tar -zcvf xxx.tar.gz
> ```
>
> 解包：
>
> ```
> tar -xvf xxx.tar
> ```
>
> 解包并解压缩：
>
> ```
> tar -zxvf xxx.tar.gz
> ```
>
